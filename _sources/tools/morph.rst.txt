.. _tool-morph:

morph
=====

File conversion toolkit.

**Extra:** ``uv tool install buvis-gems[morph]``

Commands
--------

morph html2md
~~~~~~~~~~~~~

Convert HTML files in a directory to Markdown.

.. code-block:: bash

    morph html2md ~/html-pages/

morph deblank
~~~~~~~~~~~~~

Remove blank pages from PDF files.

.. code-block:: bash

    morph deblank document.pdf
    morph deblank *.pdf
